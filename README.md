# AG News Classification

## Описание

Этот проект реализует задание "AG News Classification" в рамках курса "Курс по Вайбкодингу: Анализ данных с Python".

Цель проекта — загрузить датасет AG News, проанализировать распределение новостей по 4 категориям (World, Sports, Business, Tech), вычислить статистику длины текстов для каждой категории и выявить 15 наиболее характерных слов для каждой из них. Результаты анализа сохраняются в формате JSON, а распределение категорий визуализируется с помощью круговой диаграммы (PNG).

## Структура проекта
#### vibe_coding_101-Axwell
##### ├── assignment.py # Основной файл с логикой анализа данных 
##### ├── test.py # Файл с юнит-тестами для основных функций 
##### ├── README.md # Данный файл
##### ├── ag_news_sample.json # Файл с локальными данными (резервный источник)
##### ├── ag_news_results.json # Результаты анализа (генерируется при запуске)
##### └── ag_news_distribution.png # Визуализация распределения категорий (генерируется при запуске)

2.  **(Рекомендуется) Создание виртуального окружения:**

    ```bash
    python -m venv .venv

    ```

    ```bash

    ```
